# Simple HTML Resume

This project is part of the **Chai Code Cohort 2026 – Web Development Peer Review**.

It contains a **basic HTML resume** created using only core HTML elements.
The goal of this project is to practice proper HTML structure, semantic tags,
and GitHub workflow, without using CSS or JavaScript.
